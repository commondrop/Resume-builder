About

AResumeBuilder is a web-based application that allows users to create and customize their resumes with ease. Built using HTML, CSS, and JavaScript, this tool enables users to input their name, email, phone number, and work experience, while also providing styling options to choose between a classic or modern resume format.

Key features include:

  • Live Preview: Users can see real-time updates of their resume as they input their details.
  • Style Customization: The ability to switch between different resume themes.
  • PDF Generation: A built-in function that allows users to download their resume as a professionally formatted PDF.

    • This project showcases expertise in front-end development, including interactive UI design, dynamic content rendering, and JavaScript-based PDF generation.
